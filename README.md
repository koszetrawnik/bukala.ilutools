# Bolt ILU

Adobe Illustrator CEP extension built with React, TypeScript, Vite, and Tailwind.

## Setup

```bash
yarn install
yarn build
yarn dev
```

## Requirements

Enable PlayerDebugMode for development:

```bash
# macOS
defaults write com.adobe.CSXS.11 PlayerDebugMode 1
```

Then restart Illustrator and open: Window > Extensions > Bolt ILU
